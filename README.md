### Prerequisites
- Flutter SDK
- FlutterFlow account (if using the visual builder)
- Android Studio / VS Code with Flutter plugins

### Installation
1. **Clone the repository**
   ```bash
   git clone https://github.com/nuaimman02/scan_ai_flutterflow.git
   ```

2. **Navigate to the project directory**
   ```bash
   cd scan_ai_flutterflow/scan-a-i-x3wrue
   ```

3. **Install dependencies**
   ```bash
   flutter pub get
   ```

4. **Run the application**
   ```bash
   flutter run
   ```
